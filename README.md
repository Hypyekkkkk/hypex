## Começando

Este projeto requer NodeJS.

### Install
Clone this project

```bash
> git clone https://github.com/Hypyekkkkk/hypex.git
> cd hypex
```

Instale as dependências:

```bash
> npm install
```

### Usage
1. execute o bot do Whatsapp

```bash
> npm start
> Escaneie o QR Para Conectar O Bot No Whatsapp
```
